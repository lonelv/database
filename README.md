## 基于Laravel 的 database 组件开发的一个通用的数据库ORM工具。并且注释汉化、缓存可灵活拓展、不依赖于Laravel组件
目前支持MySQL，SQL Server，Postgres和SQLite。
### 使用说明


> `composer require "itxiao6/database"` 使用Composer 下载安装 Itxiao6/database
